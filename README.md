# OpenFree API 

OpenFree é uma plataforma que conecta empresas e freelancers de forma simples e segura.

## Tecnologias

* Java 21
* Spring Boot
* Maven
* H2 Database
* JPA/Hibernate
* Git & GitHub

## Funcionalidades atuais

* Cadastro de usuários
* Listagem de usuários
* API REST

## Funcionalidades futuras

* Cadastro de empresas
* Cadastro de freelancers
* Publicação de vagas
* Sistema de candidaturas
* Avaliações
* Dashboard
* Integração Angular
